# LegalOS Core Workflows

## Workflow A: Court Notice to No Missed Deadlines
Upload document -> OCR -> AI deadline extraction -> pending suggestions -> user review -> event creation -> reminder scheduling.

## Workflow B: Matter Activity to Invoice to MoMo Payment
Log work -> AI invoice draft -> user review -> create invoice -> send invoice -> portal payment -> webhook reconciliation.

## Workflow C: Client Portal Self-Service
Share updates/documents -> client login -> view matter/invoices -> audit logging -> secure messaging.
