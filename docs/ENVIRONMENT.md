# LegalOS Environment Variables

Core variables covered by the root `.env.example`:
- Database: `DATABASE_URL`, `DATABASE_POOL_SIZE`
- Redis: `REDIS_URL`
- JWT: `JWT_SECRET`, `JWT_ACCESS_EXPIRE_MINUTES`, `JWT_REFRESH_EXPIRE_DAYS`
- Storage: `STORAGE_ENDPOINT`, `STORAGE_ACCESS_KEY`, `STORAGE_SECRET_KEY`, `STORAGE_BUCKET`, `STORAGE_PUBLIC_URL`
- Search: `TYPESENSE_HOST`, `TYPESENSE_PORT`, `TYPESENSE_API_KEY`
- AI: `AI_SERVICE_URL`, `ANTHROPIC_API_KEY`
- Payments: `HUBTEL_CLIENT_ID`, `HUBTEL_CLIENT_SECRET`, `HUBTEL_CALLBACK_URL`, `PAYSTACK_SECRET_KEY`, `PAYSTACK_CALLBACK_URL`
- Notifications: `SENDGRID_API_KEY`, `SENDGRID_FROM`, `SMS_API_KEY`, `SMS_SENDER_ID`, `WHATSAPP_TOKEN`, `WHATSAPP_PHONE_ID`
- App: `ENVIRONMENT`, `FRONTEND_URL`, `CORS_ORIGINS`, `NEXT_PUBLIC_API_URL`, `NEXT_PUBLIC_APP_URL`
