# LegalOS Design System

## Color Palette
- Forest: `#0B2D1E`
- Forest Mid: `#143D29`
- Gold: `#C9963A`
- Cream: `#FAF8F3`
- Ink: `#1A1A18`
- Border: `#E8E4DA`
- Red Soft: `#C0392B`
- Green Soft: `#1D6E4A`
- Amber Soft: `#B8860B`
- Blue Soft: `#1A4A7A`

## Typography
- Display / headings: Playfair Display
- Body / UI: DM Sans
- Avoid Inter, Roboto, Arial, and system-ui as primary fonts.

## Layout Tokens
- Sidebar width: 216px
- Cards: 12px radius
- Pills: 20px radius
- Transitions: 140ms to 180ms ease
